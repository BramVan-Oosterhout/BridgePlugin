---+ BRIDGEHANDS -- display the bridge hands in a deal
Present the hands dealt in a table or around a board.

---++ Parameters
| *Parameter* | *Description* | *Default* |
| =web= |The web where the topic describing the hand is stored. | BASEWEB |
| =topic= |The topic where the description of the hand is stored. | BASETOPIC |
| =displayBoard= |Boolean requesting the hands to be displayed around a board. | =off= |
